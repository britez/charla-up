typescript:
en angular 1 estabamos usando javascript es6, en angular 2 tomamos la sugerencia de google de usar typescript, un superset de javascript que ofrece tipado y otras ventajas.
que quiere decir esto? que typescript transpila a javascript simple y elegante, que es lo que el browser termina usando.
el tipado es liviano
tenemos interfaces para describir objetos. en typescript, 2 tipos son compatibles si tienen la misma estructura interna. esto nos permite implementar una interface solo teniendo la forma que requiere, sin ningun "implement" explicito.

typescript tambien soporta la orientacion a clases que vimos en js es6, y nos deja usar interfaces o clases dependiendo del nivel de abstraccion que queramos usar.

TypeScript = ES6 + Types + Annotations

- Component-Based
Angular 2 is entirely component based. Controllers and $scope are no longer used.
ahora usamos componentes y directivas (de atributo y estructurales)
